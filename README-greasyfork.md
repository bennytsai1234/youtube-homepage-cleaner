## ✨ 主要功能

### 🧼 內容過濾引擎

| 類別 | 過濾內容 |
|:---|:---|
| 🚫 **廣告與推廣** | 影片廣告、首頁橫幅、贊助商內容、Premium 推廣 |
| 📱 **Shorts** | 首頁 Shorts 區塊、搜尋結果 Shorts、Shorts Grid |
| 🎬 **推薦干擾** | 合輯 (Mix)、新聞快報、貼文、電影推薦 |
| 📑 **播放清單** | 智慧隱藏首頁推薦播放清單 |
| 📉 **低熱度內容** | 自動過濾低觀看數影片 (閾值可自訂) |

### 🖱️ 全方位點擊優化

- 🚀 **一律在新分頁開啟**: 影片、Shorts、播放清單、頻道連結
- 🧠 **範圍覆蓋完整**: 首頁、搜尋結果、影片頁面

### 🌐 多語言支援

| 🇹🇼 繁體中文 | 🇨🇳 简体中文 | 🇺🇸 English | 🇯🇵 日本語 |
|:---:|:---:|:---:|:---:|

自動偵測語系！

### ⚙️ 可自訂設定

- ✅ 所有過濾規則均可獨立開關
- ✅ 低觀看數閾值可自訂
- ✅ 關鍵字黑名單、頻道黑名單
- ✅ 影片長度限制
- ✅ 設定匯出/匯入
- ✅ 過濾統計面板
