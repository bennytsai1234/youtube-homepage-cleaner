# Proposal: Notification Open in New Tab

## Problem
使用者回報在瀏覽 YouTube 通知時，希望能夠方便地以「新分頁」開啟影片或內容，避免中斷當前正在觀看的影片或頁面狀態。目前的通知點擊行為默認是在當前分頁導航。

## Solution
在腳本中新增邏輯，針對通知欄位 (`ytd-notification-renderer`) 內的連結進行處理。
主要方案包含：
1. 為通知項目的連結添加 `target="_blank"` 屬性，使其預設或可選擇在新分頁開啟。
2. 或者攔截點擊事件，根據使用者設定決定是否在新分頁開啟。

## Impact
- **使用者體驗**: 提升多工處理效率，瀏覽通知不再干擾當前播放。
- **功能性**: 補足 YouTube 原生介面操作的彈性。
