## ADDED Requirements

### Requirement: Visual Branding
The project documentation SHALL include a visual representation of the script's functionality to immediately communicate value to new users.

#### Scenario: First impression
- **WHEN** a user visits the repository page
- **THEN** they see a "Before vs After" comparison image at the top
- **AND** the image clearly shows the removal of ads and shorts

### Requirement: Unified English Branding
The project SHALL use "YouTube Cleaner" as the primary English brand name in documentation and script metadata.

#### Scenario: Name Consistency
- **WHEN** reading the English sections or headers
- **THEN** the name "YouTube Cleaner" is used instead of generic terms or "YouTube Purifier"

### Requirement: Engagement Prompts
The documentation SHALL include prominent Calls-to-Action (CTA) for social proof elements.

#### Scenario: Star prompt
- **WHEN** a user reads the installation or intro section
- **THEN** a visible prompt encourages them to star the GitHub repository
